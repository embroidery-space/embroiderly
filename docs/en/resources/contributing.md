# Contributing

Thank you for your interest in contributing to Embroiderly!
We welcome contributions from everyone, whether you're a user with a feedback or a developer ready to code.

## General Contributions

You don't need to be a developer to help make Embroiderly better.
Here's how you can contribute:

- **Bug Reports.** Found a bug? Let us know so we can fix it.
- **Feature Ideas.** Have a suggestion? We'd love to hear your ideas.
- **Documentation.** Help us improve our guides and tutorials.
- **Translations.** Assist in making Embroiderly available in more languages.

For all inquiries, please reach out through our [contacts](./contacts).

## Development

We're actively seeking contributors to help improve Embroiderly, especially:

- **macOS Developers.** Help us bring Embroiderly to Apple platforms.
- **Frontend Developers.** Assist with UI features and improvements.
- **UI/UX Designers.** Help us create a better user experience.

If you're interested in contributing code, please visit our [GitHub repository](https://github.com/embroidery-space/embroiderly).
To get started, check out these resources:

- **[ARCHITECTURE.md](https://github.com/embroidery-space/embroiderly/blob/dev/ARCHITECTURE.md).** Understand the project's architecture and design decisions.
- **[DEVELOPMENT.md](https://github.com/embroidery-space/embroiderly/blob/dev/DEVELOPMENT.md).** Set up your development environment and learn our workflows.

We are looking forward to your contributions!
