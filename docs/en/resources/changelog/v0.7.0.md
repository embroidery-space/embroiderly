# Embroiderly v0.7.0

## New Features

### Image Import

You can now convert photos and graphics into cross-stitch patterns directly in Embroiderly.
Select **File** -> **Import** -> **Image** to open the import window.

The import window provides a live preview and multiple customization options.
The preview updates automatically as you adjust settings, showing exactly how your pattern will look before importing.

### Palette Import

Custom palette files can now be imported directly from the **Colors** panel.
Click the menu button next to the palette selector and choose **Import Palettes**.

Embroiderly supports multiple palette file formats:

- `.master` and `.user` (Pattern Maker)
- `.threads` (UrsaSoftware)
- `.rng` (Cross-Stitch Pro Platinum)
- `.json` (Embroiderly)

Imported palettes appear in the **Custom** section and become available in all patterns and the image import window.

### Stitch Symbols Management

A new **Symbolss** panel lets you assign stitch symbols to palette items.
Symbols help distinguish between similar colors, especially when printing patterns.

Access the panel in editing mode via the palette menu or right-click context menu.
The panel displays all available characters from the currently selected symbol font.
Double-click any symbol to assign it to the selected palette item.

You can import custom symbol fonts (TrueType or OpenType) by clicking the menu button and selecting **Import Fonts**.
Imported fonts appear in the **Custom** section and are available across all your patterns.

### Updated Display Settings

The palette display settings now include options specifically for stitch symbols:

- **Show stitch symbols**. Toggles the visibility of symbol characters on palette items
- **Contrasting stitch symbols**. Places symbols on a white background for better visibility against thread colors

### Palette Sorting and Rearranging

You can now organize your working palette more efficiently.

**Automatic Sorting:**
Right-click the palette list in editing mode and select **Sort By** -> **Brand & Number** to reorder colors alphanumerically by brand name and thread number.

**Manual Rearranging:**
In the editing mode, palette items become draggable now.
Click and hold any color, drag it to the desired position, and release to drop it.
This lets you group related colors together or arrange them in any custom order.

### Positional Stitch Tools

Tools for petite, half, and quarter stitches now have both dynamic and fixed position variants.

**Dynamic Stitch Tools:**
The position or direction is determined by the cursor location within the grid cell.
This is the default behavior when you select the tool.

> While using dynamic stitch tools, hold `Ctrl` to lock the position or direction to match your previous stitch.
> This maintains consistency across multiple stitches without switching to a fixed-position variant.

**Positional Stitch Tools:**
Select a specific corner or direction variant to always place stitches in that position, regardless of the cursor location.
Access these variants through the tool dropdown menu or keyboard shortcuts.

## Enhancements

- The editor state is now properly reset when switching between patterns or closing a pattern, ensuring a clean workspace.
- Fixed the pencil button in the palette footer to correctly switch between regular and editing modes.
