# Embroiderly v0.3.0

## New Features

- Introduced a Palette Catalog with the ability to add and remove thread colors from the working palette.
- Implemented the actions history (`Ctrl+Z` to undo and `Ctrl+Y` to redo actions).
- Added Fabric Properties and Grid Properties windows for customization.

## Enhancements

- Improved stithes drawing:
  - Press and hold the mouse to draw multiple stitches in a single move.
  - Added a preview of line and node stitches while drawing.
