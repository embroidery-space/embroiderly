# Embroiderly v0.5.0

## New Features

- Implemented PDF export.
- Add a built-in application updater.
- Configured file associations.
- Added support for the Open With option in a system file explorer.
- Added support for dropping patterns into the pattern editor.
- Implemented pattern auto-saving.
