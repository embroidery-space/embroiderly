# Pattern Options

Embroiderly allows you to customize various aspects of your pattern, including pattern information, fabric properties, and grid display settings.
You can access these options through the **Pattern** menu in the menu bar.

## Pattern Information

Pattern information includes metadata about your design, such as the title, author, copyright notice, and description.
This information is embedded in your pattern file and exported PDF documents.

<figure>
  <img src="/images/guide/pattern-options/pattern-info-modal.png"/>
  <figcaption>Pattern Information window showing fields for title, author, copyright, and description</figcaption>
</figure>

The Pattern Information window contains four text fields:

- **Title.** The name of your pattern.
  If left empty, the pattern is labeled as "Untitled."
- **Author.** Your name or the name of the pattern designer.
- **Copyright.** Copyright information or licensing details.
- **Description**: A longer text area for additional details about the pattern.
  You can include notes about stitch counts, recommended threads, or design inspiration.

All fields are optional, but filling them out helps you organize your patterns and provides useful information when sharing designs with others.

## Fabric Properties

Fabric properties define the physical characteristics of the fabric you'll stitch on.
These settings determine the final size of your pattern and how it appears on screen.

<figure>
  <img src="/images/guide/pattern-options/fabric-modal.png" />
  <figcaption>Fabric Properties window showing fabric count, kind, size, and color options.</figcaption>
</figure>

### Fabric Count and Kind

**Fabric Count** specifies how many stitches fit per inch of fabric.
Common values include 14, 16, 18, and 20 count.

Select your fabric count from the dropdown menu.
The count applies equally to both horizontal and vertical dimensions.

**Fabric Kind** indicates the type of fabric material.
Common options include Aida, Evenweave, and Linen.

Choose the kind that matches your actual fabric.

### Fabric Size

Fabric size determines how large your stitching area is.
You can specify the size in three different units: stitches, inches, or millimeters.

Enter the width and height values in the input fields.
Use the radio buttons on the right to switch between units.
When you change units, Embroiderly automatically converts the values based on your fabric count.

Below the input fields, you'll see a summary showing the total fabric size in all three units.
This helps you visualize the physical dimensions of your pattern.

> [!NOTE]
> If you reduce the fabric size below the current pattern dimensions, any stitches outside the new bounds will be permanently removed.

### Fabric Color

The fabric color selector displays a palette of common fabric colors.
Click any color swatch to select it.

The selected color name appears below the palette.
This color affects how your pattern appears on screen and in exported PDFs, helping you visualize how the final design will look on the chosen fabric.

## Grid Properties

The grid overlays your pattern to help you count stitches and navigate your design.
You can customize the grid's appearance by adjusting line intervals, thickness, and colors.

<figure>
  <img src="/images/guide/pattern-options/grid-modal.png"/>
  <figcaption>Grid Properties window showing major lines interval, minor line settings, and major line settings.</figcaption>
</figure>

The **Major Lines Interval** controls how often thicker, more prominent grid lines appear.
The value represents the number of stitches between major lines.

**Major Lines** are the thicker grid lines that help you identify larger blocks of stitches.
They're easier to see and help you orient within the pattern.

**Minor Lines** are the thinner grid lines that appear between major lines.
They help you count individual stitches.

> [!TIP]
> Click on the line color swatch to open a color picker.
