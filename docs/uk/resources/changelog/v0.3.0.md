# Embroiderly v0.3.0

## Нові функції

- Додано каталог палітр із можливістю додавання та видалення кольорів ниток із робочої палітри.
- Реалізовано історію дій (`Ctrl+Z` для скасування та `Ctrl+Y` для повторного виконання дій).
- Додано вікна параметрів тканини та сітки.

## Покращення

- Покращено малювання стібків:
  - Натисніть і утримуйте мишу, щоб намалювати кілька стібків одним рухом.
  - Додано попередній перегляд лінійних та вузлових стібків під час малювання.
