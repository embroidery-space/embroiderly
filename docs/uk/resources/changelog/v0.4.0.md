# Embroiderly v0.4.0

## Нові функції

- Додано елементи керування режимом відображення стібків на панелі інструментів полотна.
- Додано елемент керування для показу та приховування символів стібків.

## Покращення

- Покращено підтримку схем OXS.

## Різне

- Розроблено власний дизайн застосунку.
- Виконано перехід від пакетів `.msi` до пакетів `-setup.exe` на Windows.
