# Публікація схем

Після завершення розробки схеми ви можете експортувати її як документ PDF.
Цей формат ідеально підходить для друку, обміну з іншими користувачами або продажу ваших схем.

## Експорт у PDF

Щоб експортувати схему в PDF:

1. Відкрийте схему в редакторі.
2. Виберіть **Файл** -> **Експортувати** -> **PDF** у меню застосунку.
3. За потреби налаштуйте параметри експорту (див. розділ [Налаштування експорту PDF](#налаштування-експорту-pdf) нижче).
4. Натисніть на **Експортувати документ**, щоб створити PDF.

### Колірні режими

Ви можете створити документ PDF у двох варіантах:

- **Чорно-білий варіант**, в якому стібки відображаються лише у вигляді символів.
  > [Лінійні](./working-with-patterns#лінійні-стібки) та [вузлові](./working-with-patterns#вузлові-стібки) стібки залишаються кольоровими для кращої видимості.
- **Кольоровий варіант**, в якому стібки відображаються в їхніх реальних кольорах ниток разом із символами.

## Вміст PDF

Кожен експортований документ PDF містить:

- **Інформацію про схему.** Назва, автор, авторські права, деталі про тканину та опис.
- **Палітру ниток.** Таблицю із символами, брендами, номерами та назвами кольорів.
- **Фрагменти схеми.** Візуальні частини вашої схеми, за необхідності розділені на кілька сторінок.

Щоб побачити, як виглядає готовий документ PDF, перегляньте [приклади схем](#приклади-схем-pdf) наприкінці цього розділу.

## Налаштування експорту PDF

Налаштувати параметри експорту можна двома способами:

### Під час експорту

У вікні **Експорт PDF** є розділ **Параметри фрагментів**, де ви можете змінювати налаштування.
Це дозволить вам попередньо переглянути, як різні параметри впливають на результат, перш ніж створювати фінальний документ.

Коли налаштування вас влаштують, ви зможете:

- Натиснути на **Експортувати документ**, щоб створити PDF із поточними налаштуваннями без їх збереження.
- Натиснути на **Зберегти налаштування**, щоб зберегти параметри для майбутніх експортів, а потім натиснути на **Експортувати документ**.

### У параметрах схеми

Для окремого налаштування відкрийте **Схема** -> **Налаштування публікації** у меню застосунку.
Це відкриє спеціальне вікно, де можна налаштувати всі параметри експорту.

Ці збережені налаштування будуть використовуватися як типові щоразу, коли ви експортуєте цю схему в PDF.

## Доступні параметри експорту

### Параметри фрагментів

Фрагменти --- це частини вашої схеми, на які вона розбивається для друку великих робіт на кількох сторінках.

- **Розмір фрагментів.** Максимальні розміри кожного фрагмента в стібках (типово 30x40 хрестиків).
- **Збережене перекриття.** Кількість рядків або стовпців, що дублюються на сусідніх фрагментах (типово 3).
- **Показувати номери ліній сітки.** Відображає числа вздовж країв для позначення ліній сітки (типово увімкнено).
- **Показувати центрувальні мітки.** Відображає позначки для горизонтальних і вертикальних центральних ліній (типово увімкнено).
