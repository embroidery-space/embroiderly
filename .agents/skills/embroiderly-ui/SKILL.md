# Embroiderly UI

Embroiderly UI is a local components library (`@embroiderly/ui`, at `packages/ui/`) built on top of **Reka UI** primitives and styled with **Tailwind Variants**.

## Reference Files

Here are additional resources useful for **developing** the UI Kit.

- `./references/component-structure.md` - Load when you work on components.
- `./references/theme-structure.md` - Load when you style components.
- `./references/testing.md` - Load when you test components.
