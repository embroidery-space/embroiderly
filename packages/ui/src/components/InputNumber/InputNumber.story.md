# InputNumber

A number input element with increment and decrement buttons.

## Purpose

The `InputNumber` component provides a styled numeric input field with vertical stepper buttons for incrementing and decrementing values.

## When to Use

- Use `InputNumber` for numeric data entry.
- Use the `min`, `max`, and `step` props to constrain the input range and step increments.
- Use the `formatOptions` prop to customize value display using `Intl.NumberFormat` options (e.g., decimals, currency, percentages).
