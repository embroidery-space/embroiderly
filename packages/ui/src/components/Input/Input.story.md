# Input

A text input element for entering single-line data.

## Purpose

The `Input` component provides a styled text input field with support for leading and trailing content slots.

## When to Use

- Use `Input` for single-line text entry.
- Use the `leading` and `trailing` slots to add icons, labels, or action buttons.
- Do not use `Input` for multi-line text; use `Textarea` component instead.
