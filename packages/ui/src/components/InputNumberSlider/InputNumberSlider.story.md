# InputNumberSlider

A composite component that combines a number input with a slider for numeric value selection.

## Purpose

The `InputNumberSlider` component provides a numeric input field paired with a slider, allowing users to either type a value directly or adjust it using the slider.
Both controls stay synchronized.

## When to Use

- Use `InputNumberSlider` when users need precise numeric input with visual feedback.
- Use this component for settings that benefit from both direct input and slider adjustment (e.g., opacity, zoom level, quantity).
