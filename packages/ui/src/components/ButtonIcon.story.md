# ButtonIcon

An icon-only button with a required tooltip text.

## Purpose

The `ButtonIcon` component is designed for actions that can be represented by a single icon, such as toolbar buttons, action buttons in lists, or compact UI elements.
It combines the `Button` and `Tooltip` components to ensure accessibility.

## When to Use

- Use `ButtonIcon` for icon-only actions where space is limited.
- Use `Button` when you need a text label or more complex content.
